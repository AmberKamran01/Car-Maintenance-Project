# Car-Maintenance-Project
# Python Project

## Project Description
A Python program that helps users track car service dates and log maintenance history. The program uses only built-in Python functions with no external libraries required. Built for [AICT] project, May 2026.

## Features
- Add new service records: oil change, tire rotation, brake check
- View complete maintenance history 
- Calculate next service date based on input
- Save/load data using text files
- Simple command-line interface
